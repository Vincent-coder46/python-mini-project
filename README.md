# Student Profile Generator

A simple Python script that collects student information via user input and generates a neatly formatted profile.

## Description
This project prompts the user to enter details about themselves using Python's `input()` function. Once all information is collected, the script uses f-strings to print a clean, organized student profile, separated with "=====" dividers for readability.

## Features
- Collects user input interactively
- Uses f-strings for clean output formatting
- Displays a structured, easy-to-read student profile

## Information Collected
- Name
- Matric Number
- Level
- Age
- Department
- University
- State of Origin

## How to Run
```bash
python student_profile_generator.py
```

## Example Output
```
===============STUDENT PROFILE===============
Name:John Doe
Matric_no:20/1234
Level:300
Age:20
Department:Computer Science
University:University of Lagos
State_of_origin:Lagos
===============================================
```
