# 🐍 My Python Mini Projects

A collection of small Python projects I've built while learning. Each one focuses on a different concept or skill. This list will keep growing as I complete more projects!

## 📂 Projects

- [Student Profile Generator](./README_project1.md) — Collects a student's details (name, matric number, level, department, etc.) and displays them as a clean profile card.
- [Word Analyzer](./README_project2.md) — Analyzes a sentence or word, showing different letter cases, character counts, and letter extraction.
- [Word Replacer](./README_project3.md) — Finds and replaces specific words or phrases within a piece of text.
- [Full Name Formatter](./README_project4.md) — Takes a first, middle, and last name and prints them out in a neatly formatted way.
- [Book Info Formatter](./README_project5.md) — Cleans up messy, semicolon-separated book data and formats it into a readable sentence.
- [Simple Text Formatter](./README_project6.md) — Practices capitalizing and trimming whitespace off different strings to produce clean, properly punctuated output.
