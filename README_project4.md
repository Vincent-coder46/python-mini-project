# 🤏 Full Name Formatter

A simple Python script that takes your first, middle, and last name as input and prints them out in a neatly formatted way. Nothing fancy — just a clean, readable intro to working with user input and string formatting in Python.

## 📌 What it does

The script prompts you to enter your **first name**, **middle name**, and **last name**, then displays them together in a nicely bordered output, like this:

```
==-==-==-==-==-==-==-==-==-=-==-=-==-==-==-==-==-==
           FULL_NAME FORMATTER
==-==-==-==-==-==-==-==-==-==-==-=-==-=-==-==-==-==
FULL NAME: My name is John Michael Doe

This is a vivid way to write YOUR NAME🤏
```

## 🚀 How to run it

1. Make sure you have Python 3 installed on your machine.
2. Clone this repository:
   ```bash
   git clone https://github.com/your-username/fullname-formatter.git
   ```
3. Navigate into the project folder:
   ```bash
   cd fullname-formatter
   ```
4. Run the script:
   ```bash
   python fullname_formatter.py
   ```
5. Follow the prompts and enter your first, middle, and last name when asked.

## 🧠 What I learned / practiced

This project was a simple way to get comfortable with:
- Taking user input with `input()`
- String formatting using `.format()`
- Structuring clean, readable console output
