# Python Mini Project 7: Conditional Statements Practice

**Author:** Joy Ayoola
**GitHub:** [Vincent-coder46](https://github.com/Vincent-coder46)

A collection of 15 small Python exercises focused on `if / elif / else` logic, comparison operators, and boolean conditions (`and`, `or`, `in`).

## What This Covers

- Basic if/elif/else branching
- Chained comparisons (`x < y < z`)
- Logical operators (`and`, `or`)
- String methods (`.strip()`, `.lower()`, `.endswith()`)
- List membership checks (`in`)
- List comprehensions
- The `all()` built-in function

## Exercises

1. **Positive Number Checker** – Checks if two numbers are both positive, one positive, or neither.
2. **Order Checker** – Takes three numbers and determines if they're in increasing, decreasing, or no particular order.
3. **Palindrome Checker** – Checks if a word reads the same forwards and backwards.
4. **Equality Checker** – Compares three values and reports if all are equal, two are equal, or all are different.
5. **Triangle Angle Validator** – Checks if three angles form a valid triangle (all positive, summing to 180°).
6. **Vowel or Consonant** – Identifies whether a character is a vowel or consonant.
7. **Primary Color Checker** – Checks if three given colors are all primary colors (red, blue, yellow).
8. **Mode Selector** – Activates manual, automatic, or off mode based on input.
9. **Priority Message Checker** – Flags a message as high priority if it contains keywords like "urgent."
10. **Status Checker** – Reports if two statuses are both active, one active, or none active.
11. **Image File Checker** – Checks if a filename ends in `.png`, `.jpg`, or `.gif`.
12. **Access Level Checker** – Grants full, limited, or no access based on user role.
13. **Email Validator** – Basic check for `@` and `.` in an email address.
14. **Weekday/Weekend Checker** – Identifies if a given day is a weekday or weekend.
15. **Positive List Checker** – Checks if all numbers in a comma-separated list are greater than 0.

## How to Run

```bash
python conditionals.py
```

Each exercise runs in sequence and will prompt for input via the terminal.

## Notes

- All inputs are read as strings and cast where needed (`int()`, `float()`).
- Some exercises intentionally use `.strip()` to clean up whitespace after splitting comma-separated input.
