# Word Analyzer

A simple Python script that takes a word or sentence as input and analyzes its text properties.

## Description
This project prompts the user to enter a word or sentence using Python's `input()` function, then analyzes and displays various properties of the text — such as case variations, first/last letter, and character count — using f-strings for clean output formatting.

## Features
- Collects user input interactively
- Converts text to uppercase, lowercase, and title case
- Extracts the first and last letter of the text
- Counts the total number of characters
- Displays results in a clean, formatted layout

## Information Displayed
- Original Text
- Uppercase version
- Lowercase version
- Title Case version
- First Letter
- Last Letter
- Character Count

## How to Run
```bash
python word_analyzer.py
```

## Example Output
```
==========================================
              WORD ANALYZER
============================================

Enter a word or sentence: Machine learning

=============================================
Original Text: Machine learning
Uppercase: MACHINE LEARNING
Lowercase: machine learning
Title Case: Machine Learning
First Letter: M
Last Letter: g
Characters: 16
```
