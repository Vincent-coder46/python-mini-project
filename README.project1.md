# 🎓 Student Profile Generator

A simple Python script that collects a student's details through input prompts and displays them in a clean, organized profile format.

## 📌 What it does

The script asks for basic student information — name, matric number, level, age, department, university, and state of origin — then prints it all out as a neat, readable profile card.

```
===============STUDENT PROFILE===============
Name:John Doe
Matric_no:20/1234
Level:300
Age:22
Department:Computer Science
University:University of Lagos
State_of_origin:Lagos
===============================================
```

## 🚀 How to run it

1. Make sure you have Python 3 installed on your machine.
2. Clone this repository:
   ```bash
   git clone https://github.com/Vincent-coder46/your-repo-name.git
   ```
3. Navigate into the project folder:
   ```bash
   cd your-repo-name
   ```
4. Run the script:
   ```bash
   python student_profile.py
   ```
5. Enter your details when prompted, and your profile will be displayed.

- Using f-strings for cleaner, more readable string formatting
